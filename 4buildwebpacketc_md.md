# 4.JS Build

### Webpack

Webpack은 모듈 번들러로서, 큰 파일을 서로간의 종속성을 갖는 여러 작은 모듈로 쪼갤 수 있게 하고, 해싱을 이용해 장기적인 캐싱을 통한 성능 향상을 도우며, CSS 전처리기나 compile-to-js 파일들의 컴파일에도 사용할 수 있고, 핫 리로딩을 지원하는 등 정적 에셋 관리 전반적인 프로세스에 큰 도움이 된다.
